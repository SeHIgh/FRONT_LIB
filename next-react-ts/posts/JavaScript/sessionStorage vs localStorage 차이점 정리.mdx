| **항목**          | sessionStorage          | localStorage                  |
| --------------- | ----------------------- | ----------------------------- |
| **데이터 유지 기간**   | **브라우저 세션(탭)**이 종료되면 삭제 | 사용자가 직접 삭제하지 않는 이상 **영구 저장**  |
| **페이지 새로고침 영향** | 새로고침해도 데이터 유지됨          | 새로고침해도 데이터 유지됨                |
| **브라우저/탭 간 공유** | **같은 탭 내에서만** 데이터 유지됨   | **같은 도메인 내 모든 탭/창에서 공유 가능**   |
| **저장 용량**       | 약 5MB                   | 약 5~10MB                      |
| **사용 사례**       | 로그인 상태, 일시적 데이터 저장      | 사용자 설정, 장바구니 데이터 등 장기 저장 필요 시 |
