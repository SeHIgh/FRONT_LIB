## SVG 파일 크기 조절
```html
<svg width="32.669" height="32" viewBox="0 0 98 96" ... >
	<path ... />
</svg>
```
파일 내에서 `height, width` 으로 조절하는 것이 아닌
`viewBox` 라는 요소로 크기를 조절하게 되는데

공식 문서에 따르면  https://developer.mozilla.org/ko/docs/Web/SVG/Attribute/viewBox
`min-x`, `min-y`, `width`, `height` 순으로 viewBox 에 값을 설정 해준다지만

이를 제대로 크기 조절에 적용하고자 이해한다면
부모 요소 내 이미지의 시작위치 좌표 를 처음 두 숫자 `min-x`, `min-y` 로
처음으로 다운받았을때 원 이미지의 `width` `height` 크기를  마지막 두 숫자  `width`, `height` 로 이해하면 본인의 생각대로 이미지 크기 조정이 가능할 것 이다.

본인의 경우
Github 에서 배포하는 공식 로고를 사용하고자 다운받았는데
```html
<svg width="98" height="96" ... >
	<path ... />
</svg>
```
